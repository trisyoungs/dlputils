Counts the number of frames in a DL_POLY HIStory file

## Basic Usage

```
dlpsize <HISTORYfile>
```

Where:
+ `<HISTORYfile>` is the target name of the DL_POLY HISTORY file to be written

## Switches

None.

## Output

Number of frames in HISTORY file, printed to screen.
